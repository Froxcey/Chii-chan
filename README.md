# chii-chan

A simple anime Discord bot

## Features

- Show airing schedules
- Follow and get tagged for airing shows
- Create threads for discussions
Simple innit?

## Running

0. This project depends on BunJS
1. `bun i`
2. `cp .env.template .env`
3. Fill in your .env file
4. `bun .`
